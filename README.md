# Паркінс — розширена перша версія

Багатосторінковий статичний сайт для людей, які живуть із хворобою Паркінсона, та їхніх близьких.

## Сторінки

- `index.html` — головна сторінка.
- `first-steps.html` — перші кроки після діагнозу.
- `understand.html` — хвороба, симптоми, діагностика, ON/OFF, міфи й словник.
- `treatment.html` — лікування та роль різних фахівців.
- `daily-life.html` — побут, сон, завмирання, падіння, водіння, робота, інтимність і допоміжні засоби.
- `nutrition.html` — харчування, апетит, вага, закрепи та ковтання.
- `movement.html` — демонстраційні вправи.
- `emotional-support.html` — тривога, депресивні прояви, апатія та кризова допомога.
- `doctor-visit.html` — підготовка до консультації.
- `relatives.html` — матеріали для близьких.
- `support-ukraine.html` — офіційні українські маршрути й джерела.

## Спільні файли

- `styles.css` — дизайн і адаптивність.
- `script.js` — меню, доступність, пошук, акордеони, фільтри та модальні вікна.

## Запуск

Розпакуйте архів і відкрийте `index.html`.

Або запустіть локальний сервер:

```bash
python -m http.server 8000
```

## Медичне застереження

Матеріали є демонстраційними. Перед реальною публікацією потрібні медична редактура, юридична перевірка українських маршрутів, регулярне оновлення джерел і тестування доступності з реальними користувачами.


## Зміни в цій версії

- Назву сайту змінено на «Паркінс».
- На мобільних і планшетах горизонтальне тематичне меню приховано; усі сторінки доступні в бургер-меню.
- Сторінку `understand.html` перероблено без плиткових карток: теми згруповано у великі змістові зони з горизонтальними рядками.
- Кожна з 11 тем на сторінці «Про хворобу» відкриває власне модальне вікно.


## Остання зміна оформлення сторінки «Про хворобу»

- Фон рубрик чергується: бежевий, білий, бежевий.
- Кнопки «Читати матеріал» мають темно-зелений фон, білий текст, контрастну рамку та помітний hover-стан.
- На мобільному кнопка займає всю доступну ширину.

- У розділі «Словник» встановлено білий фон і темний текст.


## Оформлення рубрик і словника

- Прибрано світло-бежеві проміжки між фонами рубрик.
- Усі рубрики утворюють один безперервний зонований блок.
- «Словник» оформлено як білу рубрику «Як хвороба може проявлятися», але без кнопок.
- Кожен термін підкреслено зеленою смужкою фіксованої довжини 72 px.


## Оформлення блоку «Бібліотека»

- Заголовок «Бібліотека», вступний текст, інформаційне вікно та пошук об’єднано в одну білу рубрику.
- Рубрика має ті самі рамки, відступи й ширину, що й інші тематичні рубрики.
- Прибрано окремий проміжок між «Бібліотекою» та першою тематичною рубрикою.


## Бежева шапка та перевірка інформації

- На сторінці `understand.html` шапка й тематична навігація мають бежевий фон.
- Рубрика «Перевірка інформації» також оформлена на бежевому фоні.
- Зміни обмежені лише сторінкою «Про хворобу».


## Нумерація в рубриках

- У кожній тематичній рубриці нумерація починається з `01`.
- Нумерація ведеться окремо в межах кожної рубрики.


## Єдине оформлення внутрішніх сторінок

- На всіх внутрішніх сторінках шапка та зона хлібних крихт мають бежевий фон.
- Вступний блок сторінки білий.
- Наступні рубрики чергуються: бежевий / білий, без вертикальних проміжків.
- Збережено функціональні елементи: акордеони, форми, вправи, чеклісти й модальні вікна.
- Аварійні попередження залишено на окремому світло-червоному фоні.
