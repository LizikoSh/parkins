# Рух поруч — багатосторінкова версія

Кожен змістовий розділ першого етапу винесено на окрему HTML-сторінку.

## Сторінки

- `index.html` — головна сторінка й навігація.
- `first-steps.html` — перші кроки після діагнозу.
- `understand.html` — інформація про хворобу та симптоми.
- `daily-life.html` — повсякденне життя.
- `movement.html` — рух і вправи.
- `doctor-visit.html` — підготовка до лікаря.
- `relatives.html` — матеріали для близьких.
- `support-ukraine.html` — допомога в Україні.

## Спільні файли

- `styles.css` — стилі всіх сторінок.
- `script.js` — мобільне меню, доступність, пошук, акордеони, фільтри й модальні вікна.

## Запуск

Розпакуйте архів і відкрийте `index.html`.

Для локального сервера:

```bash
python -m http.server 8000
```

Після запуску відкрийте `http://localhost:8000`.

## Важливо

Медичні тексти є демонстраційними. Перед публікацією їх мають перевірити профільні фахівці. Контакти та умови програм допомоги в Україні потрібно заповнити й регулярно оновлювати.
