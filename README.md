# Рух поруч — мінімальна версія сайту

Статичний адаптивний сайт першого етапу для людей, які живуть із хворобою Паркінсона, та їхніх близьких.

## Файли

- `index.html` — структура та демонстраційний контент.
- `styles.css` — адаптивний дизайн і режими доступності.
- `script.js` — меню, пошук, фільтри, акордеони, модальні вікна та друк пам’ятки.

## Як запустити

1. Розпакуйте архів.
2. Відкрийте `index.html` у браузері.

Для локальної розробки зручніше запустити простий сервер:

```bash
python -m http.server 8000
```

Потім відкрийте `http://localhost:8000`.

## Важливо перед реальною публікацією

- Увесь медичний контент має перевірити профільний лікар.
- Контакти, державні програми та маршрути отримання допомоги потрібно регулярно оновлювати.
- Потрібні окремі сторінки політики конфіденційності, редакційних стандартів і зворотного зв’язку.
- Варто провести тестування доступності за WCAG 2.2 та тестування з реальними користувачами.
