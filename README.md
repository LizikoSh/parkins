# Рух поруч — розширена перша версія

Багатосторінковий статичний сайт для людей, які живуть із хворобою Паркінсона, та їхніх близьких.

## Сторінки

- `index.html` — головна сторінка.
- `first-steps.html` — перші кроки після діагнозу.
- `understand.html` — хвороба, симптоми, діагностика, ON/OFF, міфи й словник.
- `treatment.html` — лікування та роль різних фахівців.
- `daily-life.html` — побут, сон, завмирання, падіння, водіння, робота, інтимність і допоміжні засоби.
- `nutrition.html` — харчування, апетит, вага, закрепи та ковтання.
- `movement.html` — демонстраційні вправи.
- `emotional-support.html` — тривога, депресивні прояви, апатія та кризова допомога.
- `doctor-visit.html` — підготовка до консультації.
- `relatives.html` — матеріали для близьких.
- `support-ukraine.html` — офіційні українські маршрути й джерела.

## Спільні файли

- `styles.css` — дизайн і адаптивність.
- `script.js` — меню, доступність, пошук, акордеони, фільтри та модальні вікна.

## Запуск

Розпакуйте архів і відкрийте `index.html`.

Або запустіть локальний сервер:

```bash
python -m http.server 8000
```

## Медичне застереження

Матеріали є демонстраційними. Перед реальною публікацією потрібні медична редактура, юридична перевірка українських маршрутів, регулярне оновлення джерел і тестування доступності з реальними користувачами.
